# 체크리스트

> 이 문서는 [[Engineering Thinking for Beginners]] 스킬의 핵심 산출물입니다.

---

## 분해 원칙

```
"큰 프로젝트는 한 꺼에 다 하면 어려워요. 
작은 단계로 나눠서 하나씩 해결해나가는 게 빨라요."
```

### 원칙
- 한 작업 = **1-2시간** 내 완료 가능
- 각 작업의 결과가 **바로 확인** 가능
- 의존성이 **명확한 순서**
