# Smart Video Speed — Browser Extension

## Project Overview

웹브라우저에서 재생중인 비디오의 속도를 사용자가 조절할 수 있는 **브라우저 확장**.

## Problem Statement (Framing Q1 — 해결하는 문제)

### Pain Point
- YouTube 라이브(실시간 영상)에서 기존 속도 조절 익스텐션이 **라이브도 일반 영상처럼 속도를 강제** → 버퍼링 반복 →用户体验 최악
- 라이브 영상 앞부분(재생시간이 마이너스인 구간)을 보고 싶을 때 seek하면同样的 버퍼링 문제

### 해결해야 할 것 (3가지 규칙)
1. **일반 영상:** 사용자 정의 속도로 항상 재생
2. **라이브 영상 (재생):** 사용자 정의 속도로 재생 시작 → 버퍼링 전에 자동으로 1배속으로 부드럽게 전환
3. **라이브 영상 (seek):** 즉시 사용자 정의 속도로 재생 → 실시간 approaching → 버퍼링 전에 미리 1배속 전환

### 핵심 메커니즘
- YouTube 라이브는 `playbackRate` 변경 시 버퍼링 발생
- 라이브의 "현재 시점"은 `currentTime`이 마이너스 영역 → 점점 0에 수렴
- 버퍼링 감지 → 1배속으로 전환 타이밍 조절
