# Hermes Agentをガチで使い倒すためのGitHubレポ10選

트윗 내용: 세션 간 메모리 유지, 경험에서 스스로 스킬을 만들어 성장하는 AI 에이전트 Hermes Agent를 제대로 활용하기 위한 GitHub 레포 10선.

## 1. Hermes Agent (본체)
Nous Research 공식 코어 리포지토리. MIT 라이선스.
https://github.com/NousResearch/hermes-agent

## 2. Hermes-Wiki
Hermes Agent 소스코드를 해설하는 커뮤니티 Wiki. 구현 이해에 효과적.
https://github.com/cclank/Hermes-Wiki

## 3. Atlas (hermes-ecosystem)
에코시스템 지도. 100+ 도구/스킬 조감, RAG 검색 지원.
https://github.com/ksimback/hermes-ecosystem

## 4. Control Interface
셀프호스트 대시보드. 멀티 에이전트/장시간 작업/메모리를 한 화면에서 관리.
https://github.com/xaspx/hermes-control-interface

## 5. Skill Factory
태스크 회고로 새 스킬 자동 생성/추가. 에이전트가 스스로 도구를 제작.
https://github.com/Romanescu11/hermes-skill-factory

## 6. Maestro
로컬 멀티 에이전트 협업 도구. Codex/Claude Code/Gemini 크로스로 구조화 메모리와 인계 관리.
https://github.com/ReinaMacCredy/maestro

## 7. Hermes Agent CaMeL
신뢰 경계(CaMeL)를 내장한 포크판. 프로덕션 운용 보호용.
https://github.com/nativ3ai/hermes-agent-camel

## 8. Hermes HUD
Textual 기반 TUI 모니터링 터미널. 의식의 흐름과 메모리 상태 실시간 가시화.
https://github.com/joeynyc/hermes-hud

## 9. Hermes Alpha
클라우드 환경용 Hermes Agent 배포 템플릿. Makefile과 설정 예시 포함.
https://github.com/kaminocorp/hermes-alpha

## 10. Awesome Hermes Agent
커뮤니티 엄선 플러그인/프롬프트/교육 자료 목록.
https://github.com/0xNyk/awesome-hermes-agent
