# vibe-coding-beginner-methodology

NotebookLM report와 Q&A를 바탕으로 생성한 정책 비교 초안이다. 주요 포인트는 데이터 소유권, 학습 기본값, 보존기간, 안전정책, 고위험 의사결정, 엔터프라이즈 통제다. 핵심 하이라이트: 바이브 코딩은 안드레이 카파시(Andrej Karpathy)가 대중화한 개념으로, 구체적인 코드 작성보다는 **개발자의 의도(Intent)**를 AI에게 전달하여 소프트웨어를 구축하는 방법론입니다.; **의도 중재(Intent Mediation):** 개발자는 ‘무엇(What)’과 ‘왜(Why)’에 집중하고, AI는 ‘어떻게(How)’에 집중하는 역할 분담입니다.; **프롬프트 기반 개발:** 문법적 정확성보다는 비즈니스 로직과 사용자 경험의 ‘바이브(느낌/비전)’를 정교하게 전달하는 것이 핵심입니다.
